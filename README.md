# WIFI-Troubleshoot-tool
Wifi troubleshoot diagnose tool from .bat file

1. Download the WifiTroubleshoot.bat file
2. open command prompt as administrator. win + R -> cmd /admin
3. Choose the correct option from the menu
4. Continue as usual with windows prompts
5. Press any key to continue
6. Select '4' menu option to exit.
7. Thank you, recommend to friends and colleagues.
